## 8. 等分布局

### 1）float + border-box + margin负值

```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        * {
            padding: 0;
            margin: 0;
        }

        .contaion {
            margin-left: -10px;
        }

        .contaion>div {
            float: left;
            width: 25%;
            box-sizing: border-box;
            padding-left: 10px;
        }

        .col1 {
            background-color: red;
        }

        .col2 {
            background-color: blue;
        }

        .col3 {
            background-color: yellow;
        }

        .col4 {
            background-color: green;
        }
    </style>
</head>

<body>
    <div class="contaion">
        <div>
            <div class="col1">1</div>
        </div>
        <div>
            <div class="col2">2</div>
        </div>
        <div>
            <div class="col3">3</div>
        </div>
        <div>
            <div class="col4">3</div>
        </div>
    </div>
</body>

</html>
```

## 9. 等高布局

### 1）table + table-cell

```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .contain {
            display: table;
        }

        .left,
        .right {
            display: table-cell;
        }

        .left {
            background-color: red;
        }

        .right {
            background-color: blue;
        }
    </style>
</head>

<body>
    <div class="contain">
        <div class="left">11111</div>
        <div class="right">
            2222222222222222222222222222222222222222222222222222222
            2222222222222222222222222222222222222222222222222222222
        </div>
    </div>
</body>

</html>
```

### 2) margin + padding

:::tip

- margin负值技巧: margin-top/bottom/left/right :  负值
- padding： 计算在实际宽度里面

:::

```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .contain {
            overflow: hidden;
        }

        .left,
        .right {
            float: left;
            width: 400px;
            padding-bottom: 9999px;
            margin-bottom: -9999px;
        }

        .left {
            background-color: red;
        }

        .right {
            background-color: blue;
        }
    </style>
</head>

<body>
    <div class="contain">
        <div class="left">
            111111
        </div>
        <div class="right">
            222222222222222222222
            222222222222222222222
            222222222222222222222
        </div>
    </div>
</body>

</html>
```

## 10. 全屏布局

### 1） 定位position:fixed

:::tip

- `overflow-y : auto`： 溢出设置滚动
- `position: fixed` : 生成绝对定位的元素，相对于浏览器窗口进行定位。

:::

```html
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="ie=edge">
  <title>全屏布局解决方案-定位position:fixed</title>
  <style>
    /*  定位 - 同时设置方位  fixed/absolute */
    * {
      margin: 0;
      padding: 0;
    }

    /* left:0;right:0; = width:100% */
    header,
    footer {
      position: fixed;
      left: 0;
      right: 0;
      height: 80px;
      background-color: rosybrown;
    }

    header {
      top: 0;
    }

    footer {
      bottom: 0;
    }

    /* left:0;right:0;top:80px;bottom:80px;   =  width:100%;height:100% - 80px - 80px */
    .content {
      position: fixed;
      left: 0;
      right: 0;
      top: 80px;
      bottom: 80px;
      background-color: royalblue;
    }

    .left {
      width: 80px;

</head>
```
