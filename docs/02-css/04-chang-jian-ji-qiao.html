    # 1. 设置input的placeholder的字体样式

    <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>设置input的placeholder的字体样式</title>
    <style>
        /* 直接设置color */
        .in1 {
            color: red;
        }

        /* 兼容 */
        .in2::-webkit-input-placeholder {
            /* Chrome/Opera/Safari */
            color: red;
        }

        .in2::-moz-placeholder {
            /* Firefox 19+ */
            color: red;
        }

        .in2::-moz-placeholder:-ms-input-placeholder {
            /* IE 10+ */
            color: red;
        }

        .in2:-moz-placeholder {
            /* Firefox 18- */
            color: red;
        }
    </style>
</head>

<body>

    <h1>前</h1>
    <input class="in1" type="text" placeholder="请设置用户名">
    <h1>后</h1>
    <!-- 设置input的placeholder的字体样式  -->
    <input class="in2" type="text" placeholder="请设置用户名">
</body>

</html>

    # 2.单行和多行文本超出省略

    <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>单行和多行文本超出省略号</title>
    <style>
        /* 省略号 4个 BFC mixin + 变量  框架 - 代码块 */
        .text1 {
            width: 200px;
            /* 超出不换行 */
            white-space: nowrap;

            /* 文本溢出，显示省略号 */
            text-overflow: ellipsis;

            /* 超出隐藏 */
            overflow: hidden;
        }

        .txt {
            display: -webkit-box;
            /* 设定垂直方向 */
            -webkit-box-orient: vertical;
            /* 设置行数 */
            -webkit-line-clamp: 2;
            /* 超出隐藏 */
            overflow: hidden;

            /* 
            text-overflow: ellipsis;
            word-break: break-all; 
            */
        }
    </style>
</head>

<body>
    <!-- 单行文本省略号 -->
    <p class="text1">
        庭院深深，不知有多深？杨柳依依庭院深深，不知有多深？杨柳依依庭院深深，不知有多深？杨柳依依，...

    # 4. 全屏布局，左右边距

    ​ 绝对定位和固定定位时，同时设置 left 和 right 等同于隐式地设置宽度。

    <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>定位同时设置方位情况</title>
    <style>
        * {
            margin: 0;
            padding: 0;
        }

        span {
            /* 全屏布局 - 京东+小米-移动分类页 200px - 子适应 */
            /* width:200px;  100% - 20px  calc() */
            border: 1px solid red;
            height: 100px;
            /* 100% - 200 -200 */
            position: absolute;
            left: 20px;
            right: 20px;
        }
    </style>
</head>

<body>
    <!-- 绝对定位和固定定位时，同时设置 left 和 right 等同于隐式地设置宽度  -->
    <span>1</span>
</body>

</html>

    # 7. 隐藏滚动条或更改滚动条样式

    <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>隐藏滚动条或更改滚动条样式</title>
    <style>
        .scroll-container {
            width: 500px;
            height: 150px;
            border: 1px solid #ddd;
            padding: 15px;
            overflow: auto;
        }

        .scroll-container .row {
            margin: 0;
            line-height: 1.5;
        }

        .scroll-container::-webkit-scrollbar {
            width: 8px;
            background: white;
        }

        .scroll-container::-webkit-scrollbar-corner,
        /* 滚动条角落 */
        .scroll-container::-webkit-scrollbar-thumb,
        .scroll-container::-webkit-scrollbar-track {
            border-radius: 4px;
        }

        .scroll-container::-webkit-scrollbar-corner,
        .scroll-container::-webkit-scrollbar-track {
            /* 滚动条轨道 */
            background-color: rgba(180, 160, 120, 0.1);
            box-shadow: inset 0 0 1px rgba(180, 160, 120, 0.5);
        }

        .scroll-container::-webkit-scrollbar-thumb {
            /* 滚动条手柄 */
            background-color: #00adb5;
        }
    </style>
</head>
