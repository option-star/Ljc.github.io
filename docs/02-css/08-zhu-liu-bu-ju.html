    # 3） 圣杯布局

    <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        * {
            padding: 0;
            margin: 0;
        }

        .container {
            padding: 0 200px;
        }

        .center {
            float: left;
            width: 100%;
            height: 100vh;
            background-color: red;
        }

        .left,
        .right {
            float: left;

            margin-left: -200px;
            right: -200px;
        }
    </style>
</head>

<body>
    <div class="container">
        <div class="center"></div>
        <div class="left"></div>
        <div class="right"></div>

    效果

    分析

    • 给外围的container加上padding：0 220px 0 220px
    • 将main部分放在最初的位置，加载最早，为了实现这样，只有将三者都采用float：left,为了调整两边栏的位置，给边栏再加上一个position:relative (因为相对定位后面会用到）
    • main部分因为需要自适应，width:100%占满宽度，因此这个时候边栏都在下面一行。 (opens new window)
    • 这时因为main已经占满了，为了将left拉倒最左边，需要使用margin-left:-100% (opens new window)
    • 这时right还停留在原来的位置，让right向左移动一个身位，marign-left:-200px，right就可以来到上方的最右边。 (opens new window)
    • 这时left会覆盖main的左端，对left使用使用相对定位left:-220px，同理right也要相对定位还原right:-220px
